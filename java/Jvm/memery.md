# JVM 的内存结构
jvm 的内存结构分为两部分：

- ###### 能被所有线程访问的内存数据
    > Method Area (方法区)
    > Heap (堆)
- ###### 每个线程所私有的数据
    > Java Stack (栈)
    > Program Counter Register (程序计数器)
    > Native Method Stack (原生方法栈)
